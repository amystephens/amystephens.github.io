SITE PLAN

Site Name

The name of this website will be Idaho Weather Now and it will focus on helping local residents get up to date information about the weather.

Site Purpose

The purpose of this website will be to provide daily weather information for the people living in the southeast corner of Idaho.

Site Domain

The domain for the website will be IdahoWeatherNow.com. This name clearly depicts the purpose of visitng the website and is short enough for most users to remember.
